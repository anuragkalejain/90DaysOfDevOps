Linux commands for Process Management:
  1. ps ---- Shows current runing processes
  2. top ---- Shows real time resource consumption by processes
  3. systemctl ---- Used to manage services on the system
  4. kill ---- Used to terminate a running process
  5. bg ---- Shows all the background processes
  6. nohup ---- Runs a process in background

Linux commands for General:
  1. cd ---- Used to navigate in the system
  2. mkdir ---- Used to make new directories
  3. pwd ---- Prints the current working directory
  4. rm -rf ---- Removes the directories recursively
  5. touch ---- Used to create a regular file
  6. cat ---- Used to cread a file
  7. vi/vim ---- Used to edit a file
  8. ls ---- Lists the content of the directory
  9. su ---- Used to switch between users
  10. cp ---- Used to copy file and directory
  11. mv ---- Used to move or rename file and directory

Linux commands for Networking:
  1. ip a ---- Shows interface information
  2. ip route ---- Shows the routing table of the system
  3. ping ---- Checks ICMP connectivity with another machine
  4. curl ---- Used to tranfer data to or from another system
  5. nslookup ---- Used to resolve IP/hostname using DNS
